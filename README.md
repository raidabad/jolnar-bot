🌸 مشروع جلنار (Golnar Bot)
مشروع ذكاء اصطناعي مخصص لمتجر "جلنار" للعناية بالشعر، يعمل بأسلوب صنعاني أصيل ويقدم استشارات مبيعات دقيقة.

🚀 المميزات
الذكاء الاصطناعي: يعتمد على نموذج Gemini Pro لتقديم استشارات مبيعات احترافية.

الأسلوب الصنعاني: يتبنى لهجة ودودة ومميزة (يا غالية، نورتينا، من عيوني).

إدارة ذكية: يربط بين بيانات العملاء في Supabase ونوع الخدمة (فرد أو كوافير).

دعم الوسائط: نظام تنبيه ذكي للتعامل مع الرسائل الصوتية.

🛠 المتطلبات التقنية
Python 3.x

Flask (للويب)

Google Generative AI (لمحرك الذكاء)

Supabase (قاعدة بيانات العملاء والمنتجات)

Gunicorn (للتشغيل على سيرفرات الإنتاج)

⚙️ الإعدادات (Environment Variables)
يجب إضافة المتغيرات التالية في منصة الاستضافة (Render):

SUPABASE_URL: رابط مشروع Supabase الخاص بك.

SUPABASE_KEY: مفتاح الـ API الخاص بـ Supabase.

GEMINI_API_KEY: مفتاح الـ API الخاص بـ Google AI Studio.

🚀 التشغيل
هذا المشروع مصمم ليعمل كـ Web Service على منصة Render.

Build Command: pip install -r requirements.txt

Start Command: gunicorn app:app
